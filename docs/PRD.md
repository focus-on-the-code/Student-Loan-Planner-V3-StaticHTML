# Student Loan Repayment Plan Estimator PRD

This repository implements a privacy-first, static federal student-loan repayment estimator. Requirements include quick and detailed inputs, complete plan comparison, source-backed eligibility/rules, monthly simulation records, accessible chart alternatives, print output, session recovery, undo/reset/delete controls, and committed verification artifacts.

The source PRD is preserved in `Student_Loan_Repayment_Plan_Estimator_V3-StaticHTML_PRD.md`; this file is the maintained implementation checklist for future agents.
