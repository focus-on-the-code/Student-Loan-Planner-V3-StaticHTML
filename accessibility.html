Accessibility

Accessibility statement

The app is designed toward WCAG 2.2 Level AA using semantic HTML, visible focus states, keyboard-accessible controls, live status updates, and print-friendly output.

  • All estimator controls are native form elements.
  • Results are available in semantic tables as well as the SVG chart.
  • Status updates use an aria-live region.
  • The layout reflows for small screens and supports browser print.
